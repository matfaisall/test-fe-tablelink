# React + Vite

## How to using this apps

1. Clone this repo
```json
https://github.com/matfaisall/test-fe-tablelink.git
```

2. install all dependencies

for yarn
```json
  yarn
```
or using npm 
```json
npm install
```

3. start this project

for yarn 
```json
yarn dev
```
or using npm
```json
npm run dev
```
